# Kamasutra Guide - iOS App

A native iOS app built with SwiftUI featuring:

- **57 positions** with professional cartoon illustrations, descriptions, categories, difficulty levels, benefits, tips, and variations
- **6 couple games** (Truth or Dare, Would You Rather, Passion Dice, Quiz, Challenges, Roulette)
- **10 discovery features** (Romantic Quotes, Mood Tracker, Date Night Generator, Massage Timer, Bucket List, Playlists, Love Language Test, Compatibility Quiz, Achievements)
- **Favorites system** with persistent storage
- **Dark mode** with rose/pink/purple gradient theme
- **5 tabs**: Positions, Games, Favorites, Info, Discover

## Requirements

- Xcode 15+
- iOS 17.0+
- Swift 5.0+

## Getting Started

1. Clone the repo
2. Open `KamasutraGuide.xcodeproj` in Xcode
3. Select an iOS simulator (e.g., iPhone 15)
4. Press ⌘R to build and run

## Content Language

All content is in Romanian.
